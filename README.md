# tableau-pages-app

Application React/Vite prête pour GitHub Pages.

## Lancer en local

```bash
npm install
npm run dev
```

## Publier sur GitHub Pages

1. Créer un repo public nommé `tableau-pages-app`
2. Pousser ce projet sur la branche `main`
3. Dans GitHub: Settings → Pages → Source = GitHub Actions
4. Le site sera publié à l'adresse :

`https://Mathias-Hoffmann.github.io/tableau-pages-app/`

## Important

Si le nom du repo change, mets à jour `base` dans `vite.config.js`.
